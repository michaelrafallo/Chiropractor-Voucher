Sends lead data to HighLevel and generates a custom voucher image for chiropractic services.

== Description ==

The Chiropractor Voucher plugin lets you capture leads, send their info to GoHighLevel (HighLevel CRM), and instantly generate a personalized chiropractic voucher image.

This is ideal for chiropractors running lead-generation campaigns or offering limited-time promos through landing pages, Facebook ads, or websites.

**Key Features:**

* Lead capture form
* Send data to HighLevel via webhook/API
* Auto-generate a unique voucher image with user details
* Optional: email the voucher or integrate with PDF download
* WooCommerce integration for selling or redeeming vouchers

== Installation ==

1. Upload `chiropractor-voucher` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Set up your HighLevel Webhook/API key in plugin settings.
4. Use `[voucher]` shortcode on any page or post.
5. Customize your voucher design in the template editor (optional).

== Frequently Asked Questions ==

= What is HighLevel and why is it used? =

HighLevel is a CRM and marketing automation platform. This plugin sends leads to your HighLevel workflows, allowing follow-ups, automations, and contact tracking.

= Can the voucher image be customized? =

Yes, the voucher design is customizable via the plugin's settings or template files.

= Does it support WooCommerce? =

Yes. You can optionally sell or redeem vouchers using WooCommerce.
